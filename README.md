# 🏠 House Price Prediction using Machine Learning

## 📌 Project Overview

This project predicts house prices using Machine Learning algorithms based on features like area, bedrooms, bathrooms, parking, and furnishing status. The project includes data preprocessing, visualization, model training, evaluation, and deployment.

---

## 🚀 Features

- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- Correlation Heatmaps & Visualizations
- Regression Model Training
- House Price Prediction
- Model Evaluation
- Streamlit Web Application
